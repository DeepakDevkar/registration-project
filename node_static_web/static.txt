# Simple Node.js Application to Serve Static Website

This example uses:

* Node.js
* Express.js

It will:
✅ Create a Node.js server
✅ Serve a static HTML website

---

# Step 1: Create Project Folder

Create folder:

```bash id="l0khpk"
mkdir static-website
```

Move inside folder:

```bash id="yrc0r6"
cd static-website
```

---

# Step 2: Initialize Node Project

```bash id="vh9d3u"
npm init -y
```

This creates:

```text id="wl5ycs"
package.json
```

---

# Step 3: Install Express

```bash id="jlwmfx"
npm install express
```

---

# Step 4: Create Folder Structure

Create files:

```text id="vjlwmw"
static-website/
│
├── public/
│   ├── index.html
│   └── style.css
│
├── server.js
└── package.json
```

---

# Step 5: Create `server.js`

```javascript id="4azek8"
const express = require('express');

const app = express();

const PORT = 3000;

// Serve static files
app.use(express.static('public'));

app.listen(PORT, () => {
    console.log(`Server running at http://localhost:${PORT}`);
});
```

---

# Step 6: Create `public/index.html`

```html id="pn0klj"
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Static Website</title>

  <link rel="stylesheet" href="style.css">
</head>

<body>

  <div class="container">

    <h1>Welcome to Node.js Website</h1>

    <p>This website is served using Node.js and Express.</p>

  </div>

</body>
</html>
```

---

# Step 7: Create `public/style.css`

```css id="qrfkha"
body{
    font-family: Arial;
    background:#f2f2f2;
    text-align:center;
}

.container{
    margin-top:100px;
}

h1{
    color:blue;
}
```

---

# Step 8: Run Server

```bash id="xyccql"
node server.js
```

Output:

```text id="7w9gmb"
Server running at http://localhost:3000
```

---

# Step 9: Open Browser

Visit:

```text id="c1jlwm"
http://localhost:3000
```

You will see your static website.

---

# How It Works

```text id="prjlwm"
Browser Request
      ↓
Node.js Server
      ↓
Express Serves Files
      ↓
HTML/CSS Displayed
```

---

# Important Line

```javascript id="jlwmv8"
app.use(express.static('public'));
```

This tells Express:

```text id="jlwm8z"
Serve all files inside the public folder
```
